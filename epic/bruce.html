Find Me Beer! - Persona 1

  • Name: Bruce Benning
  • Type: Power User
  • Tech: Very tech savvy with an iPhone 11, iPad 7 and an office desktop PC HP brand with 3.5 Ghz Intel processor and 16Gb RAM.
  • Occupation: Banker
  • Needs/Goals/Features: Bruce wants to be able to find new beers based on his specific tastes and save them in a list for later.
  • Frustrations: Bruce has used other beer apps that only have commercial beer brands. He also doesn’t like beer apps that do not let him save favorite beers.
  • User Story: As a power user he wants to be able to save his favorite beers he finds on the app.

  • Use Case: Bruce wants to use the Find Me Beer app to add beer he likes to his favorite beers list.
  • Precondition: User has an FMB account and needs to login.
  • Postcondition: User has beers listed inside favorites list.
  • Interaction Flow:
    1. User needs to select “find beers” icon to find new beers to add or navigate to.
    2. App will display list beers to choose from.
    3. User wants to make beer a favorite and will click favoring icon to enable favorite,
    4. App will send request to the server.
    5. Server sends back recorded data and display beer added as favorite in users favorites list.
    6. User can then go to profile and select favorites. The selected beer will now show as listed beer in favorites list.
