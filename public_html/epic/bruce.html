Find Me Beer! - Persona 1

  • Name: Bruce Bennan
  • Type: Power User
  • Tech: Very tech savvy with an iPhone 11 (ios 13.1), iPad 7 (ios 13.1) and an office desktop PC HP brand with 3.5 Ghz Intel processor and 16Gb RAM (Newest version of Windows 10).
  • Occupation: Banker
  • Needs/Goals/Features: One of his goals he wants to achieve is to find the perfect beers that match his taste. He also is trying to make a collective list of his favorite beers he finds on FMB.
  • Frustrations: Bruce has used other beer apps that only have commercial beer brands. He also doesn’t like beer apps that do not let him save favorite beers.
  • User Story: As a power user he wants to be able to save his favorite beers he finds on the app.

  • Use Case: Bruce will use the FIND ME BEER APP to login and set his beer preferences by type, color, taste and ALC/BTU level. Bruce has an acquired taste and has love for craft beers, he has distinct preferences for his beer consumption that he feels must be met in order to feel satisfied.
  • Precondition: User has an FMB account and needs to login.
  • Postcondition: User has beers listed inside favorites list.
  • Interaction Flow:
    1. User will select sign up or login.
    2. App prompts user for user info Name, DOB, email.
    3. User will login to app with email and password.
    4. App will prompt user and ask for their beer preferences.
    5. User can select between style, taste, ABV, etc and then click submit.
    6. App will display list of beers based on user’s beer preferences.
    7. User will make beer a favorite and will click favoring icon to enable beer as favorite.
    8. App will send request to the server to make beer a favorite.
    9. Server sends back recorded data and display beer added as favorite in users favorites list.
    10. User can then go to profile and select favorites. The selected beer will now show as listed beer in favorites list.
